# AI Fitness Frontend

A modern React-based frontend for an AI-powered fitness application that generates personalized diet and workout plans.

## 🚀 Tech Stack
- React.js
- Tailwind CSS
- Axios
- Recharts

## ✨ Features
- User dashboard
- AI plan generation UI
- Diet & workout visualization
- Progress tracking charts
- Responsive SaaS-style UI

## 🔗 API Integration
This frontend connects to the backend API for:
- Generating AI fitness plans
- Fetching user data
- Tracking progress
